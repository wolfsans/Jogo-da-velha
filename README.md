# Jogo da Velha (Minimax)

Projeto simples em HTML/CSS/JS com IA imbatível usando Minimax.

## Rodar local
Abra o `index.html` no navegador.

## Publicar no GitHub Pages
1. Suba estes arquivos na raiz do repositório.
2. GitHub → Settings → Pages
3. Source: Deploy from a branch
4. Branch: main (ou master) / folder: root
5. Salve. Aguarde e acesse a URL gerada.
